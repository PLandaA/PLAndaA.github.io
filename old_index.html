    Ninja Tower TakeOver

    This is a Tower Defense game. Therefore, everything is controlled by an artifical intellegence. You need click on the agent icon in order to spawn other agents. But first, you need resources, so make sure to spawn the miner, which is the one on the bottom left of the screen. Once you have enough of them you can spawn more powerfull agents. The enemy is also controlled by an AI and will respond to the players choices. Made by Patricio Landa, Enrique Riveroll & Axel Ayala.

    RUN! (VR Game)

    This is a VR Horror Game made in Unity, we used the latest Unity XR plugin framework. You need to find the in order to escape the city ruins. Be carefull though, there is a monster who won´t let you escape so easily. You can run by swinging your hands back and forth or with the VR joystick. Made by Patricio Landa & Guillermo Vega. Record by Guillermo Vega.

    CyberPunk Runner

    This a Endless Runner with a cyperpunk visual style. This video game is avialable for Android devices. You need to touch the screen in order to move up and down the player. More enemies will appear as the score increases.

    FPS with OpenGL

    This game was created with OpenGL in Visual Studio, it´s a first person shooter. In order to win, you need to destroy all the capsules by shooting them. A power-up will appear somewhere in the map, this will boost your movement speed.
